Tradingview-Webhooks-Bot

Tradingview-webhooks-bot is a lightweight framework for managing webhook alerts, generated from tradingview.com (no affiliation). Webhook alerts can be managed headless (without GUI) or include the use of a web interface. Utilizing events and actions, users can build powerful but simple executions of trades, logging of alerts or running custom python scripts. Using the built-in manage.py makes managing tradingview-webhooks-bot easy for beginners and advanced users alike, allowing for quick deployment and scaling of existing strategies.

Note: In order to receive webhooks from tradingview.com, you must be have a public facing IP address, with the port :5000 open. It is recommend to run this app with a dedicated host. However, ngrok is recommended if you are running this app on a local/personal computer.
